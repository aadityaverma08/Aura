# Aura | AI Celestial Guide

Aura is a high-precision Vedic Astrology platform that combines classical astronomical calculations with the Google Gemini AI to provide personalized life insights, daily forecasts, and direct astrological guidance.

## 🌟 Key Features

- **Global Birth Charting:** Pinpoint accuracy using coordinates fetched from any city worldwide.
- **Aura AI Chat:** A direct-response astrological guide powered by Gemini, grounded in your personal house placements.
- **Dynamic Daily Forecast:** Real-time energy flow analysis based on current planetary transits.
- **Vedic Calculations:** Detailed planetary degrees and house positions calculated using the `astronomy-engine`.

## 🚀 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Configure Environment:**
   Create a `.env.local` file and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

3. **Run the Portal:**
   ```bash
   npm run dev
   ```

4. **Navigate:**
   Open [http://localhost:3000](http://localhost:3000) to begin your celestial consultation.
